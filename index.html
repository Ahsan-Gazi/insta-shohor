Differenciate between local Storage and session storage

Local storage and session storage are almost same but there are differences between them in some cases

First of all local storage data doesn't delete after closing the browser or browser tab but session storage data instantly deletes after the browser or tab is closed..

Local storage data must delete by the user but session storage data automatically delete after the browser tab is closed..

Most probably local storage and session storage support string data type

Local storage and session storage can keep the data maximum 5mb

User can block local storage and sesison storage

Local storage has no auto expiry option but session storage has it

Local storage has the capacity to clear or delete JS only, on the other hand Session Storage can clear or delete JS and applying Automatic process
